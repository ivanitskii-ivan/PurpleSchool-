# 🎓 PurpleSchool Landing Page

Учебный проект — современный адаптивный лендинг для образовательной платформы PurpleSchool. Реализован с акцентом на качественную вёрстку, семантическую структуру HTML и интерактивный пользовательский интерфейс.

## 🚀 Основные решения

- **Адаптивная вёрстка** — корректное отображение от 320px до широкоформатных мониторов (Flexbox, CSS Grid)
- **Кастомный видеоплеер** — собственная обёртка с управлением play/pause и отслеживанием события `ended` через JavaScript
- **Мобильное меню** — бургер-меню с поддержкой клавиатурного управления и блокировкой прокрутки фона при открытии
- **Web Accessibility (a11y)** — семантические теги, атрибуты `aria-label`, `aria-hidden`, `aria-expanded`
- **Оптимизация** — загрузка скриптов через `defer` для ускорения рендеринга страницы

## 🛠 Стек

- HTML5 (семантическая разметка)
- CSS3 / Sass (переменные, media queries)
- JavaScript ES6+ (DOM API, событийная модель)
